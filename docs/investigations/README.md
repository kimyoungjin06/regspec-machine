## Structure
- `streams/<stream_name>/ongoing.md`: 해당 stream의 단일 활성 작업 문서.
- `streams/<stream_name>/note.md`: 채택된 결정/규칙.
- `streams/<stream_name>/archive/`: 실패/보류/대체 실험.
- `registry/`: stream/결정/archive 인덱스.
- `legacy/pre_lifecycle/`: 재구조화 이전 문서 보관 구역.
